# Cloud Portfolio Website

I made this portfolio website while learning web development and cloud basics.

It is built using simple HTML, CSS and JavaScript and deployed using Cloudflare Pages through GitHub.

## Live site
https://my-website-3uj.pages.dev

## What it does
- Basic portfolio page about me
- One interactive button using JavaScript
- Hosted online using Cloudflare Pages

## What I learned from this
- How websites are deployed from GitHub
- Basic CI/CD idea (push code → auto update site)
- How static websites work on the cloud
- Some basic DOM manipulation in JavaScript

## Tech used
- HTML
- CSS
- JavaScript
- Git + GitHub
- Cloudflare Pages
